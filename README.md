# FC26 Elite Tracker Bot

## Important
1. Disable Privacy Mode from BotFather.
2. Make the bot admin in the group if needed.
3. Start the bot in private once before using private reports.
4. Best add flow is replying to the player's message then sending `.add`.

## Local run
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
